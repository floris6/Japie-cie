\chapter{Introduction} \label{chap:introduction}

This document is a condensed installation guide for setting up a local development environment for the J.A.P.I.E.-cie websites. It is intended purely as a practical guide to get your local setup working, and does not cover the broader scope of the full J.A.P.I.E.-cie Reference.

If you are looking for information on any of the following topics, please refer to the full J.A.P.I.E.-cie Reference instead:
\begin{itemize}
    \item Programming languages and frameworks (HTML, CSS, JavaScript, PHP, CakePHP)
    \item Git usage and GitLab workflows
    \item Setting up and configuring the production VPS
    \item Deploying and managing websites on the VPS
    \item Matomo analytics installation and configuration
    \item Raspberry Pi setup and maintenance
    \item Maintenance procedures for the VPS, WordPress, Matomo, and Raspberry Pis
    \item Implementation details of the Japie website, Git webhook, Let's Encrypt, and Pigeonhole
\end{itemize}

\section{Structure of this document}

This document is split into two parts. The first part, Getting Started, covers the basics you need to know before setting up your local environment.

The second part, Installation Guides, walks you through the installation and configuration of Git, a local web server and local copies of the J.A.P.I.E.-cie websites, so that you can develop and test changes on your own machine before deploying them.
